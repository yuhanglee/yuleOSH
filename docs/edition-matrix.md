## 版本分界原则

### 1. 社区版 (MIT) 定位
- **目标用户**: 个人开发者、开源爱好者、评估试用
- **核心哲学**: 零门槛上手 — 无需注册、无需信用卡、无需 License 谈判
- **可用范围**: `pip install yuleosh` 即可获得的基础能力
- **分包限制**: 3 个项目，每月 10 次 Pipeline 运行
- **缺少的关键能力**: 多租户、全管线自动化、SaaS 管理面板、合规包
- **推荐路径**: 社区版 → 体验满意 → 升级 Pro

### 2. SaaS Pro (¥999/月) 定位
- **目标用户**: 嵌入式团队（5–50 人），需要 ASPICE 合规与 CI/CD 自动化
- **核心哲学**: 一价全包，不按人头计费，零摩擦升级
- **可用范围**: 云 SaaS + Docker 自托管双模式
- **包含的关键能力**:
  - 无限项目 + 无限 Pipeline
  - SDD → DDD → TDD 全管线
  - 三层 CI/CD + QEMU SIL
  - 并行 4-Agent AI Code Review
  - 一键 ASPICE 合规包
  - 多租户隔离 + 插件市场
- **缺少的关键能力**: HIL 硬件在环、私有化 K8s 部署、SAML SSO、专属 SLA

### 3. Enterprise (¥99,800/月起) 定位
- **目标用户**: 中大型企业，有私有化部署、高级安全、合规认证需求
- **核心哲学**: 私有化 + 可定制化 + 专属 SLA
- **可用范围**: 客户自有基础设施 (K8s Helm)
- **包含的关键能力**:
  - 私有化部署 + K8s Helm
  - HIL 适配器定制 (Vector/dSPACE/自定义)
  - SAML / LDAP SSO + RBAC
  - SOC 2 / ISO 27001 合规
  - 专属工程师 + 4h SLA
  - 自定义 CI 门禁配置
- **可选附加包**: ASPICE 咨询 (+¥298K/年)

### 4. ASPICE 咨询附加包 (+¥298K/年)
- **适用对象**: 正在或计划进行 ASPICE CL1–CL3 认证的企业
- **包含服务**: 3 次现场检查、定制证据包、差距分析、预评估模拟、整改辅导
- **依赖前提**: Enterprise 订阅
- **投资回报**: 传统咨询周期压缩 50%–60%

## 常见分界疑问

> **Q: 社区版能商用吗？**
> A: 可以。MIT 许可证允许任何形式的商业使用，包括内部商业项目。但社区版不包含合规证据包、多租户管理、SaaS Dashboard 等企业级能力。

> **Q: Pro 和社区版的分水岭是什么？**
> A: 核心分水岭是 **多租户 + 全管线 + 合规包**。社区版能用基础 Pipeline（Spec→Code→Test），但没有 SDD→DDD→TDD 管线、没有合规证据包、没有团队协作功能。

> **Q: Enterprise 和 Pro 的分水岭是什么？**
> A: 核心分水岭是 **HIL + 私有部署 + SSO + SLA**。Pro 适合 SaaS 模式或 Docker 自托管场景，但大规模企业需要私有 K8s、HIL 硬件连接、SAML SSO 和专属 SLA 保障。

> **Q: ASPICE 咨询包可以单独购买吗？**
> A: 不行。ASPICE 咨询包是 Enterprise 的附加服务，依赖于 yuleOSH 平台的合规证据生成能力。如果仅需咨询服务，可以联系我们了解合作伙伴推荐。
