## 1. 本轮成果

### 1.1 LLM Client 基础设施

**文件**: `src/llm/client.py`
**负责人**: 小克 👨‍💻

- OpenAI 兼容 API 客户端，基于 `urllib` 零依赖
- 环境变量配置: `LLM_API_KEY` / `LLM_BASE_URL` / `LLM_MODEL`
- 自动 fallback: `LLM_API_KEY` → `DEEPSEEK_API_KEY` → `OPENAI_API_KEY`
- 重试机制: 指数退避 `1s → 2s → 4s`，可配 `retries`
- 超时控制: `timeout` 参数直传 urlopen
- 全面错误处理: HTTP 400/429/503、无效 JSON、null content、空 choices
