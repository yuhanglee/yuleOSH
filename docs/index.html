核心功能

嵌入式开发，一个平台搞定

从需求定义到审计合规，yuleOSH 覆盖嵌入式开发全生命周期

OpenSpec 需求管理

基于 SHALL/SHOULD/MAY 语法的结构化需求定义。自动追溯矩阵，从需求到代码、测试的双向覆盖，支持增量变更（Delta）管理。

AI Agent 编排流水线

三大 Agent（小明/Hermes/Claude）自动编排：审查需求→生成设计→编写代码→自我测试→代码审查→生成报告。全自动智能协作。

SDD → DDD → TDD 管线

从软件设计文档到详细设计文档再到测试驱动开发，逐层自动化推进。设计即代码，测试即文档，确保全过程可追溯。

三层 CI/CD 流水线

Layer 1 开发验证 → Layer 2 集成验证 → Layer 3 系统验证。逐层推进，与 ASPICE 标准对齐（SWE.4 / SWE.5 / SWE.6）。

一键 ASPICE 合规包

自动生成合规审计证据包：追溯矩阵、覆盖率报告、审查日志。一键打包下载，直接用于 ASPICE 评估或客户审计。

Docker 自托管 / SaaS

一行命令 Docker 部署，数据完全掌控在您手中。同时提供 SaaS 云版本，零部署成本，快速上手体验。

常见问题

FAQ

yuleOSH 适用于哪些嵌入式场景？
yuleOSH 面向各类嵌入式系统开发团队：汽车电子（ASPICE 合规需求）、IoT 设备、工业控制、消费电子等。只要有严格的需求管理、自动化测试和合规审计需求，yuleOSH 都能显著提升效率。
和 CI/CD 工具（Jenkins/GitLab CI）有什么区别？
yuleOSH 不只是 CI/CD，它是覆盖全开发流程的平台。从需求定义（OpenSpec）、AI 设计开发、三层 CI/CD 流水线，到 ASPICE 审计证据包一键生成。传统 CI/CD 只解决"持续集成"，yuleOSH 解决"从需求到交付的端到端自动化"。
数据安全性如何保障？
Pro 版支持 Docker 自托管部署，所有数据保存在您自己的服务器上。Enterprise 版提供私有部署和 Helm Chart 支持。SaaS 版数据加密存储，定期备份。所有传输使用 TLS 加密。
如何开始使用？
免费版无需注册，直接访问 Dashboard 即可体验。如需自托管，执行 docker run -p 8080:8080 yuleosh/server 即可启动。完整文档请查看 GitHub 仓库。
