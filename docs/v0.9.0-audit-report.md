## 🔄 Pipeline 异步架构

```
POST /api/v1/pipeline/run  →  ThreadPoolExecutor.submit()
                               │
                    ┌──────────▼──────────┐
                    │  _run_pipeline_job() │
                    │  status: queued→run  │
                    │  → ci.run.run_all()  │
                    │  status: passed/fail │
                    └─────────────────────┘
                    
GET /api/v1/pipeline/status/{id}  →  get_job_status()
```

## 💳 支付流程

```
User clicks "Upgrade"  →  POST /api/v1/stripe/checkout
                            →  create_checkout_session()
                            →  Redirect to Stripe Checkout
                            
Stripe callback  →  POST /api/v1/stripe/webhook
                      →  handle_stripe_webhook()
                      →  Store.upsert_subscription()
                      →  Store.update_org_tier()
```
