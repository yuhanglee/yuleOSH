# yuleOSH 产品化 Sprint — 任务拆解

> 目标: 打通 SaaS 注册→Trial→付费→Onboarding 闭环，快速上市

---

## Track α: 小克 👨‍💻 (前端+后端工程)

| # | 任务 | 工时 | 说明 |
|:-|:-----|:----:|:-----|
| α-01 | 打通注册→Trial→付费全流程 | 2d | 注册页 → Stripe Checkout → 订阅管理 → 降级提示 |
| α-02 | 新用户 Onboarding 向导 | 1.5d | Dashboard 首屏引导：创建项目 → 运行第一条 Pipeline |
| α-03 | 定价页 (Pricing Page) | 1d | 三栏对比: Free / Pro / Enterprise |
| α-04 | Landing 页优化 | 1d | CTA强化 + 用户价值文案 + 注册转化漏斗 |
| α-05 | 生产部署配置 | 1d | Docker Compose 生产版 + nginx/HTTPS + 环境变量 + 部署文档 |

## Track β: 小马 🐴 (内容/规范/发布)

| # | 任务 | 工时 | 说明 |
|:-|:-----|:----:|:-----|
| β-01 | 定价策略文案 + 功能对比表 | 0.5d | 基于定价方案的 Free/Pro/Enterprise 差异化描述 |
| β-02 | Onboarding 用户指南更新 | 1d | 快速开始教程: 3步从注册到运行 Pipeline |

| β-04 | Launch Checklist | 0.5d | 发布前逐项确认清单 (域名/SSL/监控/备份/支持) |
| β-05 | 最终审查 + 验收 | 1d | α完成后的全面审查，确认发布条件 |
