# yuleOSH v1.0.1 发布说明

> **发布日期**: 2026-06-18
> **版本号**: v1.0.1 (Patch Release)

---

## 📋 变更概览

自 v1.0.0 GA 以来，Phase 0 质量基线修复完成，主要变更:

### 🐛 Bug 修复
- **Serial mock fallback**: 在无串口环境下自动fallback到mock模拟，确保CI和演示场景稳定
- **Demo pipeline timer**: 修复useReducer/useRef闭包导致状态过期问题，进度条平滑过渡
- **Server port param**: 修复服务端口参数传递不一致
- **GH Pages 部署修复**: Page路由、资源前缀(assetPrefix)、trailingSlash等多项部署问题修复
- **Progress bar**: 包含运行步骤计数，过渡动画更平滑

### 🔧 代码质量提升 (Phase 0)
- **模块拆分**: `flash.py` (646行精简)、`pack.py` (1057行精简)、`analyzer.py` (885行精简) — 大模块按职责拆分
- **spec 更新**: 需求规范全面修订 (230行新增)
- **验收矩阵更新**: RTM 对齐 (91行变更)
- **用户指南**: 新增 331 行文档
- **旧计划文件清理**: 移除已过期的 Sprint 计划文件

### 🌐 前端 & Demo
- **Product v1 → v2**: Template Gallery + SaaS Demo + AI Preview Analyzer
- **纯客户端 Demo**: 去除API依赖，独立运行
- **定价/首页更新**: 国际化支持，布局优化
- **GH Pages 静态构建**: 完全静态部署，无需后端

### 🧪 测试
- test_auth_extended: 结构优化
- test_usage: 68行重构
- 各测试文件适配模块拆分后的 import 路径

### 📦 依赖
- pyjwt: 2.8 → 2.13.0
- CI coverage gate: 80%

## 🚀 如何获取

```bash
# 通过 pip 安装
pip install yuleosh

# 通过 Docker
docker pull frisky1985/yuleosh:v1.0.1

# 从源码
git clone https://github.com/frisky1985/yuleOSH.git
cd yuleOSH
git checkout v1.0.1
pip install -e .
```
