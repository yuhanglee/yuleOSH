## 1. System Requirements

### SR-001: Hardware Abstraction
- The system SHALL provide a HAL abstraction layer for MCU peripherals
- The system SHALL support GPIO, UART, I2C, and SPI interfaces
- The system SHALL allow platform-independent application code

### SR-002: Build System
- The system SHALL support CMake-based cross-compilation
- The system SHALL support GCC ARM toolchain for target build
- The system SHALL produce both .elf and .hex output files

---

## 2. Software Requirements

### REQ-001: UART Communication
- The system SHALL implement UART driver with configurable baud rate
- The system SHALL provide interrupt and polling modes
- The system SHALL support circular buffer for RX data

### REQ-002: System Tick
- The system SHALL provide a system tick timer at 1kHz
- The system SHALL support software delay functions

- GIVEN the UART driver is initialized
