## 1. System Requirements

### SR-001: HAL Initialization
- The system SHALL initialize STM32 HAL library on startup
- The system SHALL configure system clock to 168MHz (STM32F4)
- The system SHALL initialize all required peripheral handles

### SR-002: FreeRTOS Integration
- The system SHALL integrate FreeRTOS with STM32 HAL timebase
- The system SHALL provide HAL_Delay implementation via FreeRTOS
- The system SHALL configure interrupt priorities for FreeRTOS

---

## 2. Software Requirements

### REQ-001: GPIO HAL
- The system SHALL configure GPIO pins via HAL_GPIO_Init()
- The system SHALL support EXTI interrupt on configured pins
- The system SHALL provide toggle and read primitives

### REQ-002: USART HAL
- The system SHALL configure USART via HAL_UART_Init()
- The system SHALL support interrupt-driven transmit and receive
- The system SHALL provide printf redirection via USART

## 3. Test Plan

### Scenario: LED Blink
- GIVEN the HAL is initialized and FreeRTOS is running
- WHEN a task toggles the LED GPIO
- THEN the LED SHALL change state at the configured interval
